## About the Collection

Twelve stories written in the sim-narrative genre. Each story unfolds in at least three layers:

- **Artistic layer** — event, scene, emotion.
- **Reflective layer** — inner monologue, confession, diary.
- **Systemic layer** — external view: report, log, statistics, analysis.

Some stories contain a fourth (meta-) layer — the author's path to the story, the patterns and thoughts that led to its creation.

These stories are the **living proof** that sim-narrative works. Each one was polished in dialogue with AI, each contains reflection and systemic vision.
